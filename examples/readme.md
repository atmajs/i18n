Examples
----

### Browser

- Prepair dependencies

	```bash
	$ cd i18n
	$ npm run examples_prepair
	```

- Start static server

	```bash
	$ cd i18n/
	$ atma server
	```

- Open sample in browser, e.g.: 'http://localhost:5777/examples/simple.html'
	
### NodeJS

- Prepair dependencies

	```bash
	$ cd i18n
	$ npm install
	```

- Start static server

	```bash
	$ cd i18n/
	$ node examples/server
	```

- View translation in browser, e.g.: 'http://localhost:5888/?language=de'
